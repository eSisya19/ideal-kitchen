# Ideal Kitchens Wardrobe & Hardware Ltd — Website

## Files
- `index.html` — Main website
- `css/style.css` — Stylesheet
- `js/main.js` — Interactivity

## Deploy to GitHub Pages (FREE — Recommended)

### Step 1: Create a GitHub Account
Go to https://github.com and sign up (free).

### Step 2: Create a New Repository
1. Click the **+** button → "New repository"
2. Name it: `ideal-kitchens` (or anything you like)
3. Set to **Public**
4. Click "Create repository"

### Step 3: Upload the Files
1. Click "uploading an existing file" on the repo page
2. Drag and drop ALL files/folders:
   - `index.html`
   - `css/` (folder with style.css)
   - `js/` (folder with main.js)
3. Click "Commit changes"

### Step 4: Enable GitHub Pages
1. Go to repo **Settings** → **Pages** (left sidebar)
2. Under "Source", select **Deploy from a branch**
3. Branch: **main**, Folder: **/ (root)**
4. Click **Save**

### Step 5: Your site is live!
Wait ~2 minutes, then visit:
`https://YOUR-GITHUB-USERNAME.github.io/ideal-kitchens/`

## Deploy to Vercel (Alternative — Also FREE)

1. Go to https://vercel.com → Sign up free
2. Click "Add New Project"
3. Drag & drop the project folder OR connect your GitHub repo
4. Click Deploy
5. Your site is live at a Vercel URL instantly!
